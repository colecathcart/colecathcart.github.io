PROJECTS

DM Screen

A full-stack tool for running games of Dungeons and Dragons! Allows users to quickly lookup rules and items (or even create their own!) while simultaneously using other online resources. A NodeJS + Express backend server facilitates queries between the React client, a MongoDB database and an external REST API.

Kanban Board Task Manager

Kanban-board webapp that I developed with a team for a school project. Allows users to create and share boards, post and comment on tasks, and message teammates! React + TypeScript used for the client, NodeJS + Express & MongoDB used for the backend.

Grocery Store Database

University project that uses a simple React webapp to demonstrate the practical usage of a relational MySQL database. Tables containing everything from product quantities to employee departments are created, queried, and updated based on the user's actions!

Go Compiler

A scratch-built compiler for a simplified subset of the Go programming language. This complex project is composed of several classes which break the task down into steps and demonstrate proper Object-Oriented Programming and documentation. Made using C++.

True Randomness Generator

A research project exploring the viability of human gameplay (from the board game 'Battleship') as an entropy source for true random number generation. Our experiment evaluated both the speed and quality of random numbers produced from this source. Data processing and mathematical randomness testing was performed using Python.
