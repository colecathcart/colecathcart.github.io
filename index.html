PROJECTS

Cisco Systems Ransomware Detector

Capstone project for Cisco Systems to develop a proof-of-concept defense against the growing threat of ransomware attacks on linux. Our solution uses entropy measurements to scan the filesystem for malicious encryptions and shut down offending processes. Implemented with an Object-Oriented design in C++, with extra consideration for optimizing CPU and memory performance.

AI-powered Itinerary Builder

Full-stack web application that harnesses OpenAI's Gpt-4o mini model to create road trip itineraries. A sleek and user-friendly NextJS frontend communicates to the LLM and a MongoDB database through a RESTful NodeJS + Express backend API. Secure user authentication is achieved through firebase tokens.

DM Screen

A full-stack tool for running games of Dungeons and Dragons! Allows users to quickly lookup rules and items (or even create their own) while simultaneously using other online resources. A NodeJS + Express backend server facilitates queries between the React client, a MongoDB database and an external API. Secure user authentication with JWTs.

Go Compiler

A scratch-built compiler for a simplified subset of the Go programming language. This complex project is composed of several classes which break the task down into steps and demonstrate proper Object-Oriented Programming style. Tree-based data structures and symbol tables are used to keep track of variable scope and syntax errors. Made using C++.

Nursing Speech-to-Text Hackathon

A hackathon to develop "CareVoice", a prototype LLM-powered speech-to-text device for a team of nursing students. Uses Deepgram to generate text from recorded conversations between nurses and patients, and OpenAI's gpt-4o mini to summarize transcripts into a variety of medical documents. Implemnted with React, Spring Boot and MySQL.
